Sobre LeadCRM

O Projeto

LeadCRM é um sistema de gerenciamento de leads desenvolvido como projeto acadêmico, demonstrando as melhores práticas de desenvolvimento web seguro e profissional.

Segurança
  • Autenticação segura com Flask-Login e JWT
  • Proteção contra SQL Injection usando SQLAlchemy parametrizado
  • Proteção contra XSS com escapamento em templates Jinja2
  • Proteção contra CSRF com tokens WTForms
  • Senhas com hash bcrypt de alta segurança
Arquitetura MVC
  • Model: SQLAlchemy ORM (User, Lead)
  • View: Templates Jinja2 com Bootstrap
  • Controller: Blueprints Flask (Auth, Leads, Main)
Tecnologias Utilizadas
Backend
  • Python 3.9+
  • Flask 2.3
  • Flask-SQLAlchemy
  • Flask-Login
  • Flask-JWT-Extended
  • Flask-WTF
Frontend
  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript
  • Font Awesome Icons
Funcionalidades
  • ✓ Registro e autenticação de usuários
  • ✓ CRUD completo de leads
  • ✓ Gerenciamento de status de leads
  • ✓ Busca e filtro de leads
  • ✓ Dashboard com estatísticas
  • ✓ Perfil de usuário
  • ✓ API REST para leads
  • ✓ Interface responsiva
Desenvolvedor

Projeto desenvolvido como trabalho acadêmico para demonstrar competências em desenvolvimento web seguro e profissional com Python/Flask.
